sysdbd-timeseries-rrdtool(5)
============================
:doctype: manpage

NAME
----
sysdbd-timeseries-rrdtool - timeseries::rrdtool plugin

SYNOPSIS
--------
  LoadPlugin timeseries::rrdtool

DESCRIPTION
-----------
*timeseries::rrdtool* is a plugin which adds support for fetching time-series
data from RRD files.

CONFIGURATION
-------------
*timeseries::rrdtool* does not currently accept any configuration options. It
registers a time-series fetcher callback called *rrdtool*.
